How to validate

Short user guide

  1. Select the schema to validate the model against on the "Schema" page and press "Load schema"
  2. Once the schema has been loaded, you are redirected to the "Validate" page
  3. Select your model by pressing the "Choose file" button
  4. Press the "Submit" button. The file will be uploaded and validated and you are redirected to the "Report" page to see the results
  5. After each validation you may cut and past the results on the report page to an editor of your choice if you want to make a copy of the validation report
  6. You can validate a new file against the same schema by going back to the "Validate" page and selecting a new file to upload.

After each session, all uploaded files are deleted from the server.
